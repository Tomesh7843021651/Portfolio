## 🏗️ Architecture Overview

The application follows a **modular and scalable structure**:
src/
├── components/
│ ├── home/ # Hero, previews, CTA
│ ├── layout/ # Navbar, Footer
│ └── shared/ # Reusable UI elements
├── pages/ # Route-based pages
├── theme/ # Light/Dark system
├── motion/ # Animations & transitions
├── styles/ # Global styles


---

## ⚙️ Tech Stack

- ⚛️ React (TypeScript)
- ⚡ Vite (fast build system)
- 🎨 CSS (custom, no heavy frameworks)
- 🔥 Firebase (planned for dynamic content)
- 🌐 React Router (routing)

---

## 🌗 Theme System

A custom-built **light/dark theme engine**:

- Dark mode → default (focused, premium feel)
- Light mode → activated via toggle
- Powered by CSS variables
- Smooth transitions across entire UI
